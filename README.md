# Enterprise Network Lab

## Overview
Enterprise network topology designed using Cisco Packet Tracer to simulate a real-world corporate network.

## Features
- VLAN Configuration
- Inter-VLAN Routing
- OSPF Routing
- DHCP
- ACL Security
- SSH Remote Access
- Network Troubleshooting

## Tools Used
- Cisco Packet Tracer

## Skills Gained
- Routing & Switching
- Network Security
- Enterprise Network Design
- Troubleshooting
